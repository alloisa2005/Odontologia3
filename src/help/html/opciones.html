Opciones de Programa



  1. USUARIOS

    2. Mediante esta opción se puede dar de alta usuarios al sistema y para cada uno de ellos, se le asignan los distintos accesos permitidos para usar el software.



    • Nuevo Usuario
      •

      Pantalla donde se da de alta un nuevo usuario del sistema con los campos obligatorios en color rojo

    • Modificar Usuario
      •

      A partir del usuario seleccionado de la lisa, se puede editar los datos previamente ingresados

    • Accesos
      •

      Pantalla donde se le brindan los accesos al usuario previamente seleccionado de la lista de usuarios




  2. MÉDICOS

    3. Pantalla que muestra los médicos ingresados en el sistema

  3. BUSCAR MÉDICO

    4. Se puede buscar médicos previamente ingresados en el sistema tanto por número de cédula como por apellido. Para buscar por número de cédula se debe ingresar un número (sin puntos ni guiones) y apretamos la tecla Enter en caso de que exista el médico aparecerá en la lista. Para buscar por apellido se debe ingresar el apellido (o parte del apellido) y apretamos la tecla Enter y apareceran en la lista los médicos que tengan como apellido lo que se escribió en el campo de apellido.

    • Nuevo Médico
      •

      Pantalla donde se da de alta un nuevo médico al sistema con los campos obligatorios en color rojo

    • Modificar Médico
      •
      Seleccionado un médico de la lista se puede modificar los datos del mismo.

  4. BASE DE DATOS

    5. Pantalla que permite hacer un backup de la base de datos o cargar un backup previamente realizado

    Mediante esta pantalla se crear un archivo backup de la base de datos, como así también cargar uno previamente realizado. La ruta destino de donde quedará el archivo al crearlo lo definimos mediante la opción "Op. Generales" dentro de este mismo panel (explicado mas abajo).


  5. OPCIONES GENERALES

    6. Pantalla que permite dar de alta opciones generales necesarias o recomendables para la utilización del programa

    • Guardar
      •
      Cargando los campos de la izquierda se procede a dar de alta una opción al programa.

    • Eliminar
      •
      Seleccionando una opción de la lista de opciones, se procede a eliminar dicha opción
