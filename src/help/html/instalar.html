Parametrización Inicial
Antes de comenzar a utilizar el software (por 1era vez) se deben tener las siguientes consideraciones:
  • Ejecución de scripts

    • Ingreso de usuario generico para la utilización por 1era vez del sisema.

      • INSERT INTO public.usuario(id, apellido, celular, contrasenia, direccion, nombre, telefono, fch_nac) VALUES ('admin', 'ADMIN', '', '123456', 'DIR PRUEBA','ADMIN', '', '01/01/1970');

      INSERT INTO public.acceso(agenda, consultas, estadisticas, opciones, pacientes, pagos, reportes, usuario_id, migracion) VALUES (true, true, true, true, true, true, true, 'admin',true);


  • Cargar opciones generales

    • Ingreso de opción general donde se establece el nro. inicial de factura (si es necesario), se ingresa el nro. 111111 a modo de ejemplo.

      • INSERT INTO public.opcion(id, descripcion, valor) VALUES ('1', 'Nro. Factura Inicial', '111111');

    • Ingreso de opción general donde se establece la ruta donde se generará el archivo backup de la base de datos (a modo de ejemplo se crea la ruta "E:\BackUps\").

      • INSERT INTO public.opcion(id, descripcion, valor) VALUES ('2', 'Path BackUp', 'E:\BackUps\');
